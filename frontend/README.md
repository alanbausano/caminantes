# Los Caminantes - Frontend 🍔

The client-side application for "Los Caminantes", featuring a landing page and a loyalty rewards system.

## 🛠 Tech Stack
- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **UI**: [Material UI (MUI)](https://mui.com/), [Framer Motion](https://www.framer.com/motion/)
- **State**: [React Query](https://tanstack.com/query/latest)
- **Scanning**: [html5-qrcode](https://github.com/mebjas/html5-qrcode)

## 🚀 Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## 📂 Features
- **Responsive Landing Page**: Interactive menu and location sections.
- **Loyalty Dashboard**: Progress tracking, coupon management, and QR redemption.
- **Auth Integration**: Support for Google and Email/Password sign-in.
